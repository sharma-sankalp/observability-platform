# Grafana

## Overview

Grafana is a visualization platform used to create dashboards from multiple data sources.

---

# Features

- Dashboards
- Variables
- Alerting
- RBAC
- Folder organization

---

# Common Dashboards

- Kubernetes Cluster
- Nodes
- Pods
- Applications
- Infrastructure
- Business Metrics

---

# Benefits

- Real-time visualization
- Multiple data sources
- Flexible dashboard design
- Enterprise authentication
